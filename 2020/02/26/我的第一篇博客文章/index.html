2020-02-26
字数统计: 600字   |   阅读时长≈ 2分钟

第一步 环境搭建

1.安装Node.js

https://nodejs.org

2.安装Git

https://git-scm.com/downloads

第二步 安装Hexo

  • win系统打开终端
    • win健 + R ,输入cmd
    • 开始菜单鼠标右键,选择PowerShell(尽量以管理员模式下进行)
    • 也可以打开VS Code,按Ctrl + ` 打开终端

1.通过npm包管理工具指定网址安装cnpm

2.利用cnpm来安装全局hexo

3.验证hexo安装是否成功

第三步 创建博客

1.新建文件夹mkdir + 文件名, 文件名不需要加”

mkdir blog 就创建了文件夹名为blog的文件夹

如果在搭建博客过程中出现任何问题,大不了把文件夹删掉重来,无需担心

2.进入创建之后的文件夹

3.初始化博客(新建一个网站)

4.出现Start blogging with Hexo! 证明初始化成功

5.dir命令可以查看当前目录下的所有文件

6.hexo server命令启动服务器(可简写为hexo s)

7.可打开浏览器,在网址栏输入localhost:4000,回车键即可查看

第四步 新建文章

1.hexo new 文章名 新建一篇文章 (注意文章名需要加”)

2.命令cd source/_posts 进入资源目录

3.dir 可查看目录下新建的文章

第五步 编辑文章

1.可使用编辑器VS Code等, Markdown软件Typora等打开后缀名为.md的文件

2.编辑文章内容参考Markdown语法即可

3.编辑完成之后返回到主目录,也就是一开始新建的文件夹

4.hexo clean 清除缓存文件

5.hexo generate 生成静态文件(可简写为hexo g)

6.hexo server 启动服务器(可简写为hexo s)

7.本地搭建已经完成

第六步 部署到GitHub

1.首先登录GitHub个人账号

2.创建新的存储库

3.部署个人博客github地址

4.创建存储库

5.创建完成

6.创建git部署的插件

7.使用VS Code打开_config.yml文件

8.编辑如下内容

9.部署到远端

10.输入git账号和密码
